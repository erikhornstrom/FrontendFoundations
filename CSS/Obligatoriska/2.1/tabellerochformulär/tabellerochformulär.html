Styling av formulär

Vi kan styla inputfält på många olika sätt. Först ser vi ett exempel på ett default inputfält utan styling:

Nedan kan vi se ett exempel på ett inputfält som vi har stylat:

Denna typ av styling har vi gjort:

  • Större textstorlek med font-size: 18px
  • Ändrat textfärgen till grå med color: grey
  • Ändrat bredd med width: 30%
  • Skapat avstånd mellan text och kantlinje med padding: 10px
  • Satt en 2px solid grå kantlinje med border: 2px solid grey
  • Rundat kanterna med border-radius: 10px
  • Tagit bort den blåa outlinen genom outline: none
  • När man klickar på länken använder vi pseudoklassen focous för att göra ytterligare styling i detta tillstånd:
  • Ändra textfärgen man skriver in till vit med color: white
  • Ändra färg på kantlinjen till svart med border-color: black
  • Ändra färg på bakgrunden med background-color: #B1BFB8
