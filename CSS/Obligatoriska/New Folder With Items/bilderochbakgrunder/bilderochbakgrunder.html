Bilder och Bakgrunder

Bakgrunder

background

background är ett property som kan användas för att sätta följande andra properties:

  • background-color

    Sätter bakgrundsfärgen för ett element.

  • background-image

    Sätter en eller flera bakgrundsbilder för elementet.

  • background-position

    Sätter startpositionen för en bakgrundsbild för elementet.

  • background-size

    Sätter storleken för bakgrunden. Antingen fördefinierat med auto, cover och contain, eller med vanliga enheter.

  • background-repeat

    Bestämmer om bakgrundsbilden ska upprepas. Exempelvis om en bild är för liten för att täcka hela elementet så kan man använda repeat för att bakgrundsbilden ska upprepas där den första tar slut.

  • background-origin

    background-origin avgör hur stor yta av elementet ska täckas. Dvs om bakgrundsbilden ska täcka content (content-box), content och padding (padding-box) eller content, padding och kantlinje (border-box).

  • background-clip

    Avgör om bakgrunden ska beskäras utanför content (content-box), utanför padding (padding-box) eller utanför kantlinjen (border-box).

  • background-attachment

    Avgör om bakgrundsbilden följa med eller inte vid scrollning.
