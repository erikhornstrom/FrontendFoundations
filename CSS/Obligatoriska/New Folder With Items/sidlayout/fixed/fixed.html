Position

position:

position: bestämmer vilken positioneringsmetod som ska användas för elementet (static, relative, absolute, fixed, sticky):

static

static är default och positionerar ett element enligt det vanliga flödet på sidan så som den här paragrafen.

relative

relative sätter positioneringen relativt till elementets vanliga position med hjälp av attributen top, right, bottom och left. Exempelvis har vi satt positionen för denna paragrafen till relative och bottom: 20px.

absolute

absolute positionerar elementet relativt till närmaste "förfader" med relativ positionering. Om det inte finns någon förfader med relativ positonering så positioneras elementet med hänsyn till webbläsarfönstret. Exempelvis kan vi positionera titeln "absolute" under paragrafen trots att den är skriven ovanför i html.

fixed

fixed positionerar ett element i relation till webbläsarfönstret och är oberoende av scroll. Exemepl är texten "FIXED" som kan ses uppe i högra hörnet.

FIXED

sticky

ett element som har positionen sticky är positionerat i relation till scrollens position. Dvs att elementet har position relative fram tills det att scrollen når en viss punkt och då får den postionen fixed.

Exempel är denna text som följer med om man scrollar nedåt på sidan.

Storlek

height

Bestämmer höjden på elementet. Vi har satt höjden på denna paragraf till 100px.

width

Bestämmer bredden på elemetet. Vi har satt bredden på detta element till 100px.

max-width

Bestämmer maxbredden på ett element. Om man förstorar ett fönster kommer elementets storlek inte överskrida denna storlek. Maxbredden på detta element är satt till 600px.

max-height

Bestämmer maxhöjden på ett element. Om man förstorar ett fönster kommer elementets höjd inte överskrida denna storlek. Maxhöjden på detta element är satt tll 25px.

min-width

Bestämmer minimumbredden på ett element. Om man förminskar ett fönster kommer elementets bredd inte bli mindre än denna storlek. Minimumbredden på detta element är satt till 300px.

min-height

Bestämmer minimumhöjden på ett element. Om man förminskar ett fönster kommer elementets höjd inte bli mindre än denna storlek. Minimumhöjden på detta element är satt till 15px.

vertical-align

Bestämmer vertikal position på ett inline-element. vertical-align är satt till top. för denna bild: Man kan jämföra med denna text.
